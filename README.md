# Sistema de Fidelización de Clientes
Trabajo Práctico – Segundo Parcial  
Arquitectura Web – 2025

## Descripción
Sistema que permite registrar clientes, asignar puntos por compras según reglas, gestionar vencimientos y registrar el uso de puntos

## Tecnologías
Supabase (PostgreSQL + API REST)
Postman (pruebas CRUD y RPC)
PL/pgSQL (funciones cargar_puntos y usar_puntos)

## Contenido
/sql: scripts de creación y funciones
/postman_collection: colección Postman exportada junto a los parametros utilizados para las pruebas
/docs: PDF con capturas, resultados y URL del repositorio
